Contains all CPC event classes.

{@link org.electrocodeogram.cpc.core.api.hub.event.CPCEvent} represents the root of the event hierarchy. {@link org.electrocodeogram.cpc.core.api.hub.event.EclipseEvent}s are generated by the CPC Sensor module. {@link org.electrocodeogram.cpc.core.api.hub.event.CloneEvent}s are generated by a number of CPC modules.

An event of special interest for contributors of UI elements is the {@link org.electrocodeogram.cpc.core.api.hub.event.CloneModificationEvent}.
